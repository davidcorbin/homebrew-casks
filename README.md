# My Casks

## How do I install these formulae?

`brew install davidcorbin/casks/<formula>`

Or `brew tap davidcorbin/casks` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "davidcorbin/casks"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
